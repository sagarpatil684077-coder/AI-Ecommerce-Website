# AI Ecommerce Website

A full-stack AI-inspired ecommerce web application built for learning purposes.

## Features
- Product listing with filters and sorting
- Responsive UI using React
- Backend REST APIs with Spring Boot
- Simulated cart and checkout
- Mock product data

## Tech Stack
**Frontend**
- React (Vite)
- JavaScript
- Tailwind CSS

**Backend**
- Spring Boot
- Java
- REST APIs

## Project Status
This project uses mock data and simulated checkout flows.
No real payments or transactions are processed.
